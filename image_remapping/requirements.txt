# Image Remapping Suite - Comprehensive Toolkit

## System Requirements

- Python 3.8 or higher
- At least 4GB RAM (8GB recommended for large images)
- Modern web browser for Gradio interface

## Installation Instructions

### Standard Installation
```bash
pip install -r requirements.txt
```

### Development Installation
```bash
# Install with development dependencies
pip install -r requirements.txt
pip install pytest pytest-cov black flake8 mypy

# Install in development mode
pip install -e .
```

### Docker Installation (Optional)
```dockerfile
FROM python:3.9-slim

WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt

COPY . .
EXPOSE 7860

CMD ["python", "main.py"]
```

## Platform-Specific Notes

### Windows
- Ensure Microsoft Visual C++ redistributables are installed
- Consider using Anaconda for easier OpenCV installation

### macOS  
- May require Xcode command line tools for some dependencies
- Use Homebrew for system-level dependencies if needed

### Linux
- Most distributions include required system libraries
- Ubuntu/Debian: `sudo apt-get install python3-dev`
- CentOS/RHEL: `sudo yum install python3-devel`

## Troubleshooting

### Common Issues

1. **OpenCV Installation Problems**
   ```bash
   # Try alternative installation
   pip uninstall opencv-python
   pip install opencv-python-headless
   ```

2. **Gradio Port Conflicts**
   ```bash
   # Specify custom port
   python main.py --port 7861
   ```

3. **Memory Issues with Large Images**
   - Reduce image dimensions in config/settings.py
   - Increase system virtual memory
   - Use lower grid densities for initial testing

4. **Import Errors**
   ```bash
   # Ensure proper Python path
   export PYTHONPATH="${PYTHONPATH}:/path/to/image_remapping_suite"
   ```

### Performance Optimization

For better performance, consider installing:
```bash
# Intel Math Kernel Library acceleration
pip install mkl

# CUDA acceleration (if NVIDIA GPU available)  
pip install opencv-python-cuda

# Numba for JIT compilation
pip install numba
```
